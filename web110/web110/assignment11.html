Assignment 11

This session has taught me about embedding images, audio, and video into a web page. It also has shown me how changing one element of a web page can make it look so much more usable. I learned how to create images using a bitmap style or vector style. I learned the differences between these 2 styles, namely that vectors are more stroke-based and bitmaps are more pixel-based. I also learned how to assign a class tag to HTML elements to organize the page. In the future, I will use all of these when creating webpages to make it have more functionality. Most people would prefer pages to have imagebs, videos, and links to keep their attention. My page still looks basic, but with some more sessions I think it will look way better.
