Assignment 12

Select Statements

    SELECT * FROM CUSTOMERS;
    SELECT CustomerID FROM Customers;
    SELECT CustomerName, Address, City, Postalcode FROM Customers WHERE Country='UK';
    SELECT ContactName, CustomerName FROM [Customers] ORDER BY ContactName;
    SELECT COUNT(CustomerID) FROM [Customers];
    SELECT COUNT (CustomerID), Country FROM [Customers] GROUP BY Country;
    SELECT COUNT (CustomerID), Country FROM [Customers] GROUP BY Country ORDER BY COUNT(Country);
    SELECT COUNT (CustomerID), Country FROM [Customers] GROUP BY Country HAVING COUNT(CustomerID) Greater Than 9 ORDER BY COUNT(Country);

Insert, Update, and Delete Statements

    INSERT INTO Shippers (ShipperID, ShipperName, Phone) VALUES (4, 'On Time Delivery', '(503) 555-0123');
    UPDATE [Products] SET Price = Price + 1 WHERE Price;
    UPDATE [Products] SET Price = Price - 1 WHERE Price;
    UPDATE [Shippers] SET ShipperName = 'On-Time Delivery' WHERE ShipperID = 4;
    DELETE FROM Shippers WHERE ShipperID = 4;

This session focused on SQL and relations in databases. This lesson came at the right time because we are learning more about it in CIS 143. I learned the formatting of how to select data, insert, update, and delete from a SQL database. I initially thought we had to create a server with a database but what we did was much simpler, using a database already on Microsoft Access. In the future, understanding these basics of how to query and create databases will be critical in back end development for websites. I know that not only for websites but internally in organizations databases will have to be organized and managed.
