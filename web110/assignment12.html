Assignment 12

This session delved into JavaScript and how it functions in HTML. I've had a little experience in JavaScript but not much as all in the front end of web development. I learned how to integrate JavaScipt into my HTML using the script tags. Even the Canvas tag relies on JavaScript to style it. Previously I worked on JavaScript by itself to solve problems but integrating it into HTML is a different process. This is useful because in the future the web will still be reliant on JavaScript to keep it interactive. JavaScript is robust and useful for coding and scripting over the web. Understanding all the tools JavaScript offers to enhance HTML is only going to get more important.
